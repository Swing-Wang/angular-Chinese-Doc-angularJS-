ng里的函数组件

名称 描述
angular.lowercase

把字符串转换为小写格式。
angular.uppercase
angular.forEach

对obj 集合中的每一个元素调用iterator （迭代）函数 ，obj 集合可以是对象或者数组。 iterator（迭代） function is invoked with iterator(value, key, obj), where value is the value of an object property or an array element, key is the object property key or array element index and obj is the obj itself. Specifying a context for the function is optional.
angular.extend

Extends the destination object dst by copying own enumerable properties from the src object(s) to dst. You can specify multiple src objects. If you want to preserve original objects, you can do so by passing an empty object as the target: var object = angular.extend({}, object1, object2). Note: Keep in mind that angular.extend does not support recursive merge (deep copy).
angular.noop

A function that performs no operations. 这个函数会在使用函数式风格书写代码时十分重要。T

function foo(callback) {
  var result = calculateResult();
  (callback || angular.noop)(result);
}
angular.identity

一个返还自身第一个参数的函数。这个函数在使用函数式风格书写代码时十分有用。

angular.isUndefined

判断一个引用是否未定义（undefined）。

angular.isDefined

判断一个引用是否已定义。
angular.isObject

判断引用是否是一个Object类型。与JavaScript中的typeof 不同，nulls在此不认为是一个对象。.注意JavaScript的array是对象。
angular.isString

判断一个引用是否是String类型。
angular.isNumber

判断一个引用是否是Number类型。
angular.isDate

判断一个引用是否是一个日期类型。
angular.isArray

判断一个引用是否是 Array类型。
angular.isFunction

判断一个引用是否是Function类型。
angular.isElement

判断一个引用是否是DOM元素（或者包裹着的jQuery元素）。
angular.copy

对source创建一个深拷贝。source应当是一个对象化着数组。
angular.equals

判断两个对象或者量遏制是否相等。支持值、正则表达式、数组和对象。

angular.bind

返回一个函数。Returns a function which calls function fn bound to self (self becomes the this for fn). You can supply optional args that are prebound to the function. This feature is also known as partial application, as distinguished from function currying.
angular.toJson

将输入谢列花城一个JSON格式的字符串。以$$字符开头的属性会被去掉，因为只能angular内部使用它。

angular.fromJson

反序列化一个JSON格式字符串。
angular.bootstrap

使用这个函数手动启动angular应用。

angular.reloadWithDebugInfo

Use this function to reload the current application with debug information turned on. This takes precedence over a call to $compileProvider.debugInfoEnabled(false).
angular.injector

Creates an injector object that can be used for retrieving services as well as for dependency injection (see dependency injection).
angular.element

Wraps a raw DOM element or HTML string as a jQuery element.
angular.module

angular.module是一个可以新建、初始化和检索angular模块的全局域。 All modules (angular core or 3rd party) that should be available to an application must be registered using this mechanism.
