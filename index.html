AngularJS API 文档

欢迎来到AngularJS API文档页面。这些页面包含了AngularJS的参考资料。（版本：1.3.9-local+sha.a3c3bf3 snapshot）.

这篇文档按照包含AngularJS应用的各种组件的modules来组织。这些组件有directives， services， filters， providers， templates， 全局APIs，和模拟测试。

Angular的前缀 $和$$: 为了防止和你的代码发生命名冲突， Angular的公共对象（public)都有前缀 $，私有对象都有前缀$$。请不要在你的代码里面使用$ 或者 $$ 前缀。

Angular模块

ng (核心模块)

这个module由angular默认提供，并且包含了AngularJS的核心组件。

Directives

这是你在用模板（template）构建一个AngularJS应用时会用到的核心指令的集合。

包括以下例子： ngClick， ngInclude， ngRepeat， 等等…

Services / Factories

这是一个在你的应用中会用到的核心服务的集合。

包括以下例子: $compile， $http， $location， 等等…

Filters

ng模块里面的过滤器（filters）可以在模板数据绑定到指令或者表达式之前转换它。

包括以下例子: filter， date， currency， lowercase， uppercase， 等等...

Global APIs

核心的全局API直接绑定到angular对象上。这心核心函数在你的应用的底层操作中十分有用。

Some examples include: angular.copy()， angular.equals()， angular.element()， 等等...

ngRoute

可以在你的应用中使用ngRoute实现路由功能。ngRoute支持通过hashbang和HTML5 pushState来管理URL。

在你的应用中包含angular-route.js文件并设置ngRoute为应用的依赖。
Services / Factories 下面的应用被用来实现路由管理
  • $routeParams is 用于得到URL上的查询字段的值。
  • $route用来到达当前路由的详细位置。
  • $routeProvider 用来注册路由。
Directives The ngView 指令会在页面中显示当前路由的模板。

ngAnimate

使用ngAnimate在你的应用中实现动画效果。如果你的应用中导入了ngAnimate模块，你可以使用多种ng指令来多种动画特效。动画通过使用css变换/动画或者JavaScript回调实现。

包含angular-animate.js文件并且设置ngAnimate为你的应用的依赖便可使用。
Services / Factories 在你的指令代码中使用 $animate 来触发动画操作。
CSS-based animations 在AngularJS中遵循ngAnimate的CSS命名结构来引用css过渡/关键帧动画。一旦被定义，动画能够通过在HTML模板中引用css class触发。
JS-based animations 使用 module.animation() 去注册JavaScript动画。一旦被定义，动画能够通过在HTML模板中引用css class触发。

ngAria

使用ngAria在指令中注入 to inject common accessibility attributes into directives and improve the experience for users with disabilities.

包含angular-aria.js文件并且设置ngAnimate为你的应用的依赖便可使用。
Services

$aria服务包含附加ARIA属性到HTML上的帮助方法。

$ariaProvider用来配置ARIA的属性。

ngResource

通过REST API查询和发送数据时，可以使用ngResource模块。

导入angular-resource.js文件并且设置ngResource为应用的依赖便可使用。
Services / Factories $resource 用来定义RESTFUL对象用于与REST API交换数据。

ngCookies

在应用中使用ngCookies处理cookie。

导入 angular-cookies.js文件并且设置ngCookies 为应用的依赖便可使用。
Services / Factories 下面的服务用于管理cookie:
  • The $cookie service is a convenient wrapper to store simple data within browser cookies.
  • $cookieStore is used to store more complex data using serialization.

ngTouch

Use ngTouch when developing for mobile browsers/devices.

Include the angular-touch.js file and set ngTouch as a dependency for this to work in your application.
Services / Factories The $swipe service is used to register and manage mobile DOM events.
Directives Various directives are available in ngTouch to emulate mobile DOM events.

ngSanitize

Use ngSanitize to securely parse and manipulate HTML data in your application.

Include the angular-sanitize.js file and set ngSanitize as a dependency for this to work in your application.
Services / Factories The $sanitize service is used to clean up dangerous HTML code in a quick and convenient way.
Filters The linky filter is used to turn URLs into HTML links within the provided string.

ngMock

Use ngMock to inject and mock modules， factories， services and providers within your unit tests.

Include the angular-mocks.js file into your test runner for this to work.
Services / Factories

ngMock will extend the behavior of various core services to become testing aware and manageable in a synchronous manner.

Some examples include: $timeout， $interval， $log， $httpBackend， etc...

Global APIs

Various helper functions are available to inject and mock modules within unit test code.

Some examples inject()， module()， dump()， etc...
